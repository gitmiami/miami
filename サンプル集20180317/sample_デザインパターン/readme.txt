■『Java言語で学ぶデザインパターン入門』　付属CD-ROMについて（編集部）

　このたびは『Java言語で学ぶデザインパターン入門』をお買い上げいただき、 誠にありがとうございます。以下にこのCD-ROMの内容、ならびに利用上の注意点をまとめました。ご利用前に必ずお読みください。

●CD-ROMの内容

　このCD-ROMには、本文で紹介したサンプルプログラムが収録されています。サンプルプログラムはWindows用とUNIX用の２種類を用意しました。みなさんの環境に応じたものをお使いください。
　Windows用のサンプルプログラムは［Windows］ディレクトリに、UNIX用のサンプルプログラムは［Unix］ディレクトリに、それぞれ収録されています。

●サンプルプログラムのディレクトリ構造

　サンプルプログラムのディレクトリ構造は、以下のようになっています。


    src
    +------ Observer
    |       +------ Sample
    |       |       +------ DigitObserver.java
    |       |       +------ GraphObserver.java
    |       |       +------ NumberGenerator.java
    |       |       +------ Observer.java
    |       |       +------ RandomNumberGenerator.java
    |       |       +------ Main.java
    |       |
    |       +------ Q1
    |       |       +------ Main.java
    |       |
    |       +------ A1
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |       |
    |       +------ A2
    |       |       +       .
    |       |       |       .
    |       |       +------ Main.java
    |       
    +------ TemplateMethod
    |       +------ Sample
    |       |       .
    |       .       .
    .       .
    .

　各ディレクトリの意味は、次のとおりです。

　　src/パターン名/Sample      パターンの章で解説したサンプルプログラム
　　src/パターン名/Q☆...      パターンの章の練習問題に登場するリスト（☆は問題番号）
　　src/パターン名/A☆...      パターンの章の練習問題の解答に登場するリスト（☆は問題番号）
