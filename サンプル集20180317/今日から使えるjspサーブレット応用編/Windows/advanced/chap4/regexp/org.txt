  1.本書の特徴
━━━━━━━━
○CD-ROMと連動した実用サンプルの数々
　「XMLはすごいらしい。でも、どんな局面でXMLはつかえるの？」
　そんな疑問、モヤモヤに本書はひとつの解をご提供します。基本的な技術
　を活用した静的な画面の構築から、JavaScriptによるクライアントサイド
　スクリプト、ASP(Active Server Pages)によるサーバサイドスクリプトと、
　多岐にまたがる幅広いサンプル群は、既存書に類を見ません。
　また、全サンプルソースがCD-ROMに収録されていますので、本書の説明に
　したがっていけば、すぐに自分のPCで動作確認ができます。ライセンスフ
　リーのサンプルですから、自在にカスタマイズして、自分のサイトに取り
　こむことも可能。本書内の詳細なロジック解説が、あなたのプログラミン
　グを助けます。

○W3C標準仕様に準拠
　「XML、XMLって騒いでるけど、IEだけのしくみなんじゃないの？」
　「XMLは、自分のシステムにもとりこめるの？」
　問題ありません。
　本書は、IE5.xをベースにしたサンプルをご紹介していますが、そこで使用
　されているXSLTやxPath、DOMなどは標準化団体W3C(World Wide Web Conso-
　rtium)で策定された標準仕様を前提としています。
　皆さんにとって最も身近なアプリケーションの一つでもあるIEをベースに、
　XMLをまず「体験」し、汎用的なXMLアプリケーション開発の手がかりに---
　それが本書のポリシーです。

○XML応用事例と分析
　「実際に世間ではどのようにXMLが使われているのだろう？」
　「本当にXMLは今現在採用に値する技術なの？」
　そんな疑問をお持ちの方は多いはずです。XML導入を検討するに際しても、
　周辺の事情が分からないために二の足を踏んでいる方も少なくはないはず。
　本書では、XMLに先立つ既存技術SGML(Standard Generalized Markup 
　Language)やHTML、EDI(Elecronic Data Interchange)、RDB(Relational Data
　Base)などとの比較検証をおこなうと同時に、各種応用実例、逆の視点からの
　「XMLの落とし穴」等を交え、これまでに類を見ない精緻なXMLの実態を分析
　を試みます

○最新XML技術に準拠したリファレンス
　「サンプル集なんて、初心者だけが使う読み捨ての本だ」
　そんなイメージを持っておられる方、ちょっと待ってください。
　本書はXMLを基本に、これを取り囲む豊富な周辺技術XSLT(eXtensible 
　Stylesheet Language)、xPath(XML Path Language)、DOM(Document Object
　Model)、XMLSchemaなどのリファレンスを豊富な用例と共にご提供します。
　ひととおりの学習を終え、実際にアプリケーション開発をおこなおうとした
　とき、本書はかならず座右の書として、あなたを導いてくれるはずです。
　
○実践ノウハウを結集したコラム群
　読み物としても、充分に魅力的なコラムを数多く掲載。
　XML最新事情、本文では触れられなかったさまざまな追加TIPS。魅力的な次世
　代プログラミングモデル「.NET」などをも織り込んだその世界に、あなたは
　きっとXML以上のXMLを感じることになるでしょう。
