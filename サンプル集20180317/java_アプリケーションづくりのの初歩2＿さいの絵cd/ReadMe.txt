■ReadMe.txtファイル
　いま、ご覧のファイルです（本書のXページに掲載している「付属CD-ROMのご案内」をテキストファイルにしたものです）。

■プログラムの作成実行.pdfファイル  【 必ずお読みください！ 】

　Java実行環境のインストール用ファイルです。

　blanco Frameworkプロジェクトが開発・配布している「Eclipse 日本語化言語パック（サードパーティ版）」を適用したJava開発環境「Eclipse 3.5.1」が圧縮されています。対応OSは、Windows XP/Vista/7です。インストール方法は、本書の付録Aをご覧ください。

■sampleフォルダ
　本書に掲載されているサンプルプログラムが収録されています。本文で「09-01/ImportExample.java」と示されているプログラムコードのファイルは、「09-01S」フォルダ→「src」フォルダにImportExample.javaとして収録されています。ここに収録されているプロジェクトは、Eclipseへ取り込んで利用できます。その方法は、本書の付録Aをご覧ください。

■workspaceフォルダ
　作成したプログラムファイルを置くフォルダです。本書での学習向けに設定されています。
